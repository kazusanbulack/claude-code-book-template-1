# ブロック崩しゲーム

## 概要

このプロジェクトは、クラシックなブロック崩しゲームの実装です。プレイヤーはパドルを左右に動かしてボールを跳ね返し、画面上部のブロックをすべて破壊することを目指します。

### ゲームの特徴
- シンプルで直感的な操作
- クラシックなアーケードゲームの体験
- レスポンシブなゲームプレイ

## 起動方法

### 必要な環境
- モダンなWebブラウザ（Chrome、Firefox、Safari、Edge等）
- インターネット接続（初回のみ）

### ローカル環境での実行

#### 方法1: 直接ファイルを開く
1. リポジトリをクローンまたはダウンロード
```bash
git clone https://github.com/kazusanbulack/claude-code-book-template-1.git
cd claude-code-book-template-1
```

2. `index.html`をWebブラウザで開く

#### 方法2: ローカルサーバーを使用（推奨）
```bash
# Python 3の場合
python -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000

# Node.jsがインストールされている場合
npx http-server

# Live Serverを使用（VS Code拡張機能）
```

3. ブラウザで `http://localhost:8000` にアクセス

## 操作方法

### キーボード操作
- **← (左矢印キー)**: パドルを左に移動
- **→ (右矢印キー)**: パドルを右に移動
- **スペースキー**: ゲーム開始/ボール発射
- **P キー**: ゲーム一時停止/再開
- **R キー**: ゲームリセット

### マウス操作
- **マウス移動**: パドルがマウスの左右の動きに追従
- **クリック**: ゲーム開始/ボール発射

### ゲームルール
1. パドルでボールを跳ね返してブロックを破壊する
2. すべてのブロックを破壊するとクリア
3. ボールが画面下部に落ちるとミス
4. 3回ミスするとゲームオーバー

│   ├── game.js          # ゲームクラス（予定）
│   ├── ball.js          # ボールクラス（予定）
│   ├── paddle.js        # パドルクラス（予定）
│   ├── block.js         # ブロッククラス（予定）
│   └── utils.js         # ユーティリティ関数（予定）
├── assets/
│   ├── images/          # 画像ファイル（予定）

### 各ファイルの役割

- **index.html**: ゲームのメインページ
- **css/style.css**: ゲーム画面のスタイリング
- **js/main.js**: ゲームの初期化と全体制御
- **js/game.js**: ゲーム状態管理とループ処理
- **js/ball.js**: ボールの物理挙動と衝突判定
- **js/paddle.js**: パドルの操作と制御
- **js/block.js**: ブロックの配置と破壊処理
